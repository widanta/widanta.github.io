## 👨‍💻 Tentang Saya

Halo! Saya **I Made Widanta Abdi Nugraha**, seorang mahasiswa program studi Sistem Informasi di ITB STIKOM Bali yang memiliki minat besar pada pengembangan perangkat lunak dan desain antarmuka digital. Saya berspesialisasi dalam membangun aplikasi web yang cepat, responsif, dan mudah dipelihara.

Perjalanan teknologi saya dimulai secara autodidak, dan kini saya telah mengumpulkan pengalaman praktis melalui proyek _freelance_ dan magang. Fokus utama saya berada pada ekosistem Laravel, integrasi WordPress (WooCommerce), serta manajemen basis data MySQL.

### 💡 Keahlian & Layanan

- **Web Development:** Membangun website dinamis dan fungsional menggunakan HTML5, CSS3, JavaScript, PHP, dan **Framework Laravel**.
- **UI/UX Design:** Merancang antarmuka yang berpusat pada kenyamanan pengguna dan membuat purwarupa interaktif menggunakan **Figma**.
- **Graphic Design:** Menciptakan identitas visual merek dan aset vektor menggunakan Adobe Illustrator dan CorelDRAW.
